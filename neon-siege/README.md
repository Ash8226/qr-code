# Neon Siege

A self-contained browser platform shooter inspired by classic arcade run-and-gun games.

## Play

If GitHub Pages is enabled for this repository, play it at:

https://ash8226.github.io/qr-code/neon-siege/

## Controls

- Move: `A` / `D` or arrow keys
- Jump: `W`, up arrow, or space
- Fire: `J`
- Grenade: `K`
- Drop: `S`
- Pause: `P`

## Goal

Fight through enemy patrols and turrets, collect supplies, destroy the final signal core, and reach extraction.
